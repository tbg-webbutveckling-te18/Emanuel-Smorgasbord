Before you continue, it can be good to know what the different comparison operators does.

  1. == // Equal to
  2. === // Equal value and equal type
  3. != // Not equal to
  4. !== // Not equal value or not equal type
  5. < // Less than
  6. > // Greater than
  7. <= // Less than or equal to
  8. >= // Greater than or equal to

Keep in mind a single equal mark (=) is not a comparison operator. It is most often just used to assign values to variables etc.
