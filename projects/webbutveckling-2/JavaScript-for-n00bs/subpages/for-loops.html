The For Loops is something that can be used if you for example want to loop through items in an array. There are two types of loops that are usable. The most common one, a standard for loop, and the less common forEach.

Let's start with the regular for loop.

  1. // Example
  2. cars = ['Volvo', 'Audi', 'Koeningsegg'];

  3. for (i = 0; i < cars.length; i++) {
  4. console.log(cars[i]);
  5. }

This example above will write the car brands from the array into the browsers console on separate lines, starting with Volvo.

However, the forEach loop is a little more complex. It needs to call on a function that runs for each item in the array.

  1. // Example
  2. cars = ['Volvo', 'Audi', 'Koeningsegg'];
  3. cars.forEach(carFunction);

  4. function carFunction(car) {
  5. console.log(car);
  6. }

This forEach loop will work exactly the same as the first example.
